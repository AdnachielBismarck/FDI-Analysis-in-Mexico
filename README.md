\# Foreign Direct Investment (FDI) Analysis in Mexico



This project analyzes Foreign Direct Investment (FDI) flows in Mexico using official quarterly data.



\## Project Structure



\- `Cleanup\_Scripts/` → Data cleaning and transformation notebooks

\- `Original\_Data/` → Raw official datasets

\- `Prepared\_Data/` → (Excluded from repository due to size)



\## Notes



Prepared data files are excluded from version control and can be regenerated by running the cleanup scripts.
